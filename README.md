# ML Classification Project

## Objective
The objective of this project is to build a machine learning classification model to predict student performance using different machine learning algorithms.

---

## Dataset Used
Student Performance Dataset

The dataset contains:
- Gender
- Reading score
- Writing score
- Math score
- Lunch type
- Test preparation course
- Other student-related features

---

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Algorithms Used
1. Logistic Regression
2. Random Forest Classifier

---

## Project Steps
1. Data Loading
2. Data Cleaning
3. Data Visualization
4. Feature Engineering
5. Train-Test Split
6. Model Training
7. Accuracy Evaluation
8. Confusion Matrix Generation

---

## Evaluation Metrics
- Accuracy Score
- Precision Score
- Recall Score
- F1 Score

---

## Visualizations Included
- Histogram
- Scatter Plot
- Box Plot
- Bar Chart
- Confusion Matrix

---

## Result
The Random Forest model achieved better accuracy compared to Logistic Regression for student performance prediction.
